NVDA是什么

NonVisual Desktop Access（NVDA）是Windows上的 完全免费且开源的屏幕阅读器 （通过语音合成或盲文显示的形式，使视障者能够使用计算机），它由澳大利亚的非营利性组织 NV Access 和世界各地的贡献者共同开发。

兼容情况

支持的操作系统

  • Windows10 32/64位；
  • Windows 8/8.1 32/64位；
  • Windows7 SP1 32/64位；
  • Windows ARM64；
  • Windows Server2008 R2 SP1 及以上；

不支持的操作系统

  • Windows RT；
  • Windows 10 Mobile；

几个特殊的版本

以下列出 NVDA 的几个特殊版本，如飞确有必要，我们不建议使用，请酌情下载。

NVDA2019.2.1 （兼容旧版插件的最后一个版本）

自2019.3起，NVDA使用 Python3进行开发，多数旧版插件存在兼容性问题，目前绝大多数插件已升级完成，但如果您依赖的插件暂时找不到兼容新版 NVDA 的版本可下载最后一个支持旧版插件的NVDA 2019.2.1 安装使用。

NVDA2017.3（适用于旧版操作系统）

Windows XP 和 Vista以及Windows7（非SP1）等旧版操作系统可使用的最后一个版本是 NVDA 2017.3（不建议在新版操作系统上安装该版本）。
