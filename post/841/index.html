简介

该插件可在NVDA启动时调整Windows系统音量，防止音量过小或系统被静音时用户无法使用屏幕阅读器。

说明

  1. 安装插件；
  2. 安装后案下 NVDA + N 打开“NVDA菜单”；
  3. 找到“工具”菜单展开，选择“啟動音量設定”并回车；
  4. 根据需要设置。
