什么是 PDF，它有何优势？福昕阅读器可以支持屏幕阅读器吗？

　　PDF全称Portable Document Format，是Adobe公司开发的电子文件格式，这种文件格式可以将文字、字型、格式、颜色、版权、图像等信息独立于操作系统平台封装到一个文件中，不管是在 Windows 还是在 Mac OS 上都能保证统一的显示效果。这一特点使它成为在Internet上进行电子文档发行和数字化信息传播的理想文档格式。越来越多的电子图书、产品说明、公司文稿、网络资料、电子邮件开始使用PDF格式文件。

　　福昕阅读器是国内老牌的 PDF 阅读解决方案，在之前的版本中，福昕阅读器没有无障碍支持，视障用户基本无法使用，从9.3 版起，福昕阅读器针对 NVDA（免费开源屏幕阅读器）做了无障碍支持，NVDA 支持使用强大的浏览模式在福昕阅读器中访问PDF内容并与之交互，操作体验与 Adobe Acrobat 几乎没有区别，除了针对屏幕阅读器的朗读以外，福昕阅读器还针对低视力的用户做了放大和颜色对比度等无障碍特性，这也是笔者强烈安利这款阅读器的原因之一。 　　注意： 本文所述功能特性只推荐在NVDA和JAWS等屏幕阅读器下使用，国内屏幕阅读器暂不支持。

扫描版/文字版的 PDF 各有何特点？如何分辨呢？

　　由于PDF文件可以将若干图像封装到一个文件中，许多扫描仪是可以直接输出PDF文件的，这也就是所谓的“扫描版 PDF”的主要来源，与扫描版相对应的是文字版，顾名思义，文字版其主要内容是文字，而非图片，更严谨的说文字版可以包含文本、图像、表格，文字内容是以矢量格式存储的，而扫描版其性质是位图，内容主要是以图片形式存储的。

　　在具体使用中，文字版PDF的文字内容可以被检索，随意的选中、复制，甚至删除，即可以二次编辑。对于扫描版，我们想检索或编辑 / 复制其中的文字内容就没有那么随心所欲了，可能需要先借助 OCR 技术将图片中的文字提取出来，在视觉呈现上，扫描版的清晰度直接取决于扫描分辨率，调整缩放后可能不得不接受显示模糊或者失真的后果，而文字版几乎不存在这样的困扰，从辨别角度上，对于同一内容而言，扫描版的体积通常比纯文字版的体积大，当然这并不是一个严谨的辨别方法。嗯！还有一个分辨方法，扫描版的 PDF 通常无法直接被屏幕阅读器读取。

PDF格式如此优秀，我们如何阅读呢？

　　作为我们普通用户而言，关心的无非两方面，如何创建与如何阅读 PDF，但本文我们着重介绍基于 NVDA 屏幕阅读器的阅读解决方案，至于如何编辑/创建，可以期待后续的文章，不过，最简单的，我们可以将Word文档另存为 PDF格式文件，通常在打印前我们可以这样做来保证打印效果。不错的一点，在 Word 里进行的一些排版样式结构，当转换为 PDF 后是依然有效的。

　　说起 PDF 这种文件格式，笔者经常被问道“如何用屏幕阅读器查阅PDF”，以及“PDF对视障者是否友好”之类的问题，之所以有这样的疑问也是因为 PDF 有扫描版与文字版上的差异。

　　可以肯定的是，屏幕阅读器用户是能够阅读 PDF 文件的，至于是否友好，那要取决于我们所拿到的具体文件形式，更确切 的说要取决于创建该 PDF 的人是否进行了无障碍考量。

　　对于PDF文件本身而言，它有着非常完善的无障碍特性支持： 包括：文字可检索，表单可填写，标题/超链接等样式可导航，目录/书签可跳转，语言标记准确等无障碍特性，但以上特性需要 PDF 创建者进行矫正或添加（如使用 Adobe Acrobat Pro DC中的辅助功能检查助手）。

　　以上这些特性都非常有意义，拿其中的“语言标记”这一特性来说，它支持在一个文档中包含多种语言，支持自动语言切换的屏幕阅读器会根据此标记来切换发音角色，比如用日语语音读出日文内容，用英语语音读出英文内容，遇到中文内容也会以中文语音朗读，但据笔者所了解目前仅有 NVDA 和 JAWS 的部分合成器支持多语言自动切换。

　　说回来，笔者在前面已经介绍过，PDF可以粗略的分为扫描版/文字版，下面，我们一起来看一下具体该如何查阅这两种形式的PDF文件：

　　针对扫描版PDF的阅读方案其实万变不离其宗，受限于当今屏幕阅读器的实现机制，即文本转语音（TTS）技术，要实现语音话阅读，我们首先需要将位图内容变为文本内容，此类技术最被我们所熟知的就是 OCR (=Optical character recognition)，市面上有很多科技大厂推出了相应的云服务，如，国内的腾讯云、百度云，国外的 Google、 微软等，基于这些厂商的云服务也诞生了各种 OCR 识别工具，如，我们视障者所熟知的“白描”、 “落寞 OCR”和NVDA的“在线图像描述”插件等。对于大型的整本PDF材料，我们往往希望能够批量识别，而以上云服务一是依赖网络无法保证识别速度，二是多数存在识别次数限制，所以就不得不谈到另一款本地的软件 “ABBYY FineReader” 其可以离线识别，识别准确率在行业内也处于领先水平，基于以上，我们可以根据自己的情况，选用合适的工具。

　　对于文字版 PDF 我们便需要尽可能选用一些专用的 PDF阅读器来查阅以保证良好的阅读体验，如本文的主角—福昕阅读器，当然 NVDA 支持的PDF解决方案不只如此，还有上面提到的 Adobe Acrobat 以及 Google Chrome和 Microsoft Edge等浏览器的PDF查看器，我们在后续文章中会逐一为大家介绍。

安装福昕阅读器

  1. 从文末参考链接中下载福昕阅读器 PC 版；
  2. 下载后双击安装程序进行默认安装；
    • PS： 安装过程中有个“安装福昕高级PDF编辑器的14天免费试用版本”这里我们可以取消勾选，方法是按一次下方向键选中“不安装福昕高级PDF编辑器的14天免费试用版本”的单选按钮。
  3. 其他的可以保持默认；
  4. 安装好之后，可以在桌面上找到“福昕阅读器”打开。
  5. 打开后如果你当前开启了 NVDA 屏幕阅读器会有个辅助功能引导助手，我们可以使用推荐的默认设置，在这个辅助功能设置助手不仅包含屏幕阅读器相关的选项，还有针对低视力用户的对比度、放大镜等功能的设置选项。

阅读过程中常用的一些快捷键

以下列出一些在阅读器内常用的快捷键，这些快捷键基本上是各阅读器通用的。

  1. 前后翻页 Ctrl + PageUp/PageDown；
  2. 输入页码跳转 Ctrl + G；
  3. 在功能区中跳转 F6（与 MS Word 中功能类似，切换各个功能区）；
  4. 添加书签 Ctrl + B。

NVDA 还提供了哪些支持

NVDA在福昕阅读器下可以启用强大的浏览模式，以下是 NVDA 在浏览模式下适用的一些快捷键

当我们按下 NVDA + 空格键切换到浏览模式后可以使用以下单个字母在各个元素之间快速定位。

  • H 标题；
  • 大键盘 1 ~6，分别定位1到6级标题；
  • T 表格；
  • I 列表项目；
  • K 超链接

　　若想了解更多请参看NVDA用户指南 ➡ 第六章 浏览模式 ➡ 第一节 单键导航 中的内容。
